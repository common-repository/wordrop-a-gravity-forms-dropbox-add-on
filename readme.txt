== Description ==

### Store Gravity Forms Data in DropBox Cloud Service using the Wordrop Plugin


####Gravity Forms + Wordrop = Happy Forms Managment on the DropBox Cloud Service

This free Wordrop Add-On for Gravity Forms Allowes Saving Forms Data on the DropBox cloud.


###Originally developed by <a href="http://www.wordrop.net/">Wordrop.net</a>

== Screenshots ==

== Installation == 

1. Upload plugin files to your plugins folder, or install using WordPress' built-in Add New Plugin installer
1. Activate the plugin
